# WebUI

`webui/` は、複数動画を順次アップロードし、サーバー側で通常抽出は最大 8 件、背景除去ありは最大 3 件まで並列実行する WebUI です。

## できること

- 複数動画の一括選択
- ブラウザ側から順次アップロード
- 動画ごとの待機 / 実行 / 完了 / エラー表示
- 完了した動画から個別 zip ダウンロード
- 完了済みジョブの一括ダウンロード
- 完了 / エラーのジョブを一覧からクリア
- `muscut.py` / `muscut_with_rembg.py` のモード切替

## 起動

```bash
cd /home/idm-kurume/ghq/github.com/naomitsu-ozawa/deep_mus_cut
python webui/app.py
```

デフォルトポートは `8000` です。

## 注意

- 既存 `webui/` とは独立しています
- 全体の同時実行上限は 8 件です
- 背景除去ありのジョブは同時に 3 件までです
- アップロード動画と中間生成物は処理後に削除します
- zip はダウンロード後に削除します
