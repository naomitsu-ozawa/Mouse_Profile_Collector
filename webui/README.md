# WebUI

`webui/` は、ブラウザから複数動画を投入して抽出処理を行う WebUI です。  
通常抽出に加えて、モデル設定とモデル確認の画面を持ちます。

## できること

- 複数動画の一括選択と順次アップロード
- 通常抽出と背景除去ありの切替
- 動画ごとの待機 / 実行 / 完了 / エラー表示
- 完了した動画の個別ダウンロード
- 完了済み動画の一括ダウンロード
- 完了 / エラーのジョブを一覧からクリア
- ピント参照画像の生成と表示
- 設定画面でのモデル切替
- 設定画面での YOLO / CNN しきい値変更
- モデル確認画面での確認動画生成

## 起動

```bash
cd /path/to/deep_mus_cut
python webui/app.py
```

デフォルトポートは `8000` です。
起動すると、通常はローカルブラウザで `http://127.0.0.1:8000` を自動で開きます。

## 終了

WebUI を起動したターミナルで `Ctrl+C` を押すと終了できます。

## ローカルネットワーク越しの接続

WebUI はデフォルトで `0.0.0.0:8000` で待ち受けるため、同じローカルネットワーク内の別端末からアクセスできます。

### 1. サーバーPC の IP アドレスを確認

Linux の例:

```bash
hostname -I
```

または

```bash
ip a
```

macOS の例:

```bash
ifconfig
```

### 2. 別端末のブラウザからアクセス

```text
http://<サーバーPCのIPアドレス>:8000
```

### 3. 接続できない場合の確認

- サーバーPC と接続端末が同じネットワークにいるか確認してください
- サーバーPC 側で `python webui/app.py` が起動したままになっているか確認してください
- OS のファイアウォールで `8000` 番ポートが遮断されていないか確認してください
- `0.0.0.0` ではなく `127.0.0.1` だけで起動していないか確認してください

## 画面

- `処理画面`
  複数動画の投入、進捗確認、ダウンロード、ピント参照画像表示
- `設定`
  頭部検出モデル、画像分類モデル、YOLO しきい値、CNN しきい値の設定
- `モデル確認`
  指定した 1 本の動画に対して確認動画を生成し、ブラウザで再生またはダウンロード

## 処理画面の要点

- 動画は複数本を追加選択できます
- サーバー側では通常抽出を最大 `8` 件、背景除去ありを最大 `3` 件まで同時処理します
- 完了したジョブは個別に zip ダウンロードできます
- 完了済みジョブはまとめて zip ダウンロードできます
- ダウンロード後の zip はサーバー側から削除されます
- `先頭動画で参照画像を表示` は、選択済み動画の先頭 1 本だけを使います

## モデルの置き方

- 標準モデル
  既存の固定パスをそのまま使います
- カスタム頭部検出モデル
  `muscut_models/custom_yolo/` に配置します
- カスタム画像分類モデル
  `muscut_models/custom_cnn/` に配置します

Linux / Windows では以下を候補として見ます。

- YOLO: `muscut_models/custom_yolo/` 配下の `*.pt`
- CNN: `muscut_models/custom_cnn/` 配下で `saved_model.pb` を含むフォルダ

macOS では以下を候補として見ます。

- YOLO: `muscut_models/custom_yolo/` 配下の `*.mlmodel`
- CNN: `muscut_models/custom_cnn/` 配下の `*.mlmodel`

## 設定の考え方

- 設定画面
  モデルとモデル由来のしきい値を設定します
- 処理画面
  抽出ジョブごとの設定を入力します
- モデル確認画面
  モデルの見え方を 1 本の動画で確認します

ピント閾値はモデル設定ではないため、設定画面には置いていません。  
処理画面の値をモデル確認画面と共有します。

## モデル確認の後始末

- モデル確認動画はダウンロード後に削除します
- モデル確認画面を閉じた場合も、確認動画と関連ファイルを削除します
- 確認動画はブラウザ再生しやすい形式へ変換して返します

## 設定ファイル

- WebUI の設定は `webui/settings.json` に保存されます
- 保存対象は、頭部検出モデル、画像分類モデル、YOLO しきい値、CNN しきい値です
- 保存した設定は、新しく開始する抽出ジョブとモデル確認ジョブから反映されます
